Menerapkan Perintah Gambar 2D pada AutoCAD

Penguasaan perintah dasar yang wajib dikuasai untuk meningkatkan produktivitas desain Anda

Jelajahi Perintah

Tujuan Pembelajaran

01
Mengenal antarmuka dasar AutoCAD

Pengenalan dan navigasi pada ribbon, drawing area, dan panel lainnya

02
Menggunakan perintah-perintah dasar untuk menggambar

Mengaplikasikan LINE, CIRCLE, RECTANGLE, dan perintah dasar lainnya

Perintah Dasar AutoCAD

LINE (L)

Fungsi:

Membuat garis lurus

Langkah:
  1. Klik ikon Line
  2. Klik titik awal dan titik akhir
  3. Tekan Enter untuk selesai menggambar garis lurus
Shortcut: L Draw

CIRCLE (C)

Fungsi:

Membuat lingkaran

Langkah:
  1. Klik ikon Circle
  2. Klik titik pusat
  3. Masukkan radius atau klik titik kedua
Shortcut: C Draw

RECTANGLE (REC)

Fungsi:

Membuat persegi panjang

Langkah:
  1. Klik ikon Rectangle
  2. Klik titik sudut pertama
  3. Klik titik sudut kedua untuk membuat persegi panjang
Shortcut: REC Draw

MOVE (M)

Fungsi:

Memindahkan objek

Langkah:
  1. Pilih objek yang akan dipindahkan
  2. Klik ikon Move
  3. Klik titik base point
  4. Klik titik tujuan
Shortcut: M Modify

COPY (CO)

Fungsi:

Menggandakan objek

Langkah:
  1. Pilih objek yang akan dicopy
  2. Klik ikon Copy
  3. Tentukan base point
  4. Klik titik tujuan
Shortcut: CO Modify

ROTATE (RO)

Fungsi:

Memutar objek terhadap titik pusat

Langkah:
  1. Pilih objek → Enter
  2. Tentukan titik putar
  3. Masukkan sudut rotasi atau klik arah putaran
Shortcut: RO Modify

ARC (A)

Fungsi:

Membuat garis lengkung atau busur (bagian dari lingkaran).

Langkah:
  1. Ketik A → Enter
  2. Klik titik awal
  3. Klik titik tengah busur
  4. Klik titik akhir busur
Shortcut: A Draw

POLYLINE (PL)

Fungsi:

Membuat garis terhubung (segmen garis atau lengkung) sebagai satu kesatuan objek.

Langkah:
  1. Ketik PL → Enter
  2. Klik titik awal
  3. Klik titik selanjutnya untuk membuat segmen
  4. Tekan Enter untuk menyelesaikan
Shortcut: PL Draw

STRETCH (S)

Fungsi:

Menarik atau memanjangkan bagian objek

Langkah:
  1. Gunakan crossing window (kanan ke kiri)
  2. Ketik S → Enter
  3. Tentukan base point → tarik ke arah yang diinginkan
Shortcut: S Modify

EXTEND (EX)

Fungsi:

Memperpanjang garis hingga menyentuh batas tertentu

Langkah:
  1. Ketik EX → Enter dua kali
  2. Klik objek yang akan diperpanjang
Shortcut: EX Modify

TRIM (TR)

Fungsi:

Memotong bagian objek yang tidak diperlukan

Langkah:
  1. Ketik TR → Enter dua kali
  2. Klik bagian objek yang ingin dipotong
Shortcut: TR Modify

EXPLODE (X)

Fungsi:

Memecah objek kompleks (misalnya block, polyline) menjadi bagian-bagian terpisah

Langkah:
  1. Pilih objek
  2. Ketik X → Enter
Shortcut: X Modify

MIRROR (MI)

Fungsi:

Membuat bayangan cermin dari suatu objek

Langkah:
  1. Pilih objek → Enter
  2. Tentukan dua titik garis mirror
  3. Ketik Y atau N untuk menghapus objek asli
Shortcut: MI Modify

ARRAY (AR)

Fungsi:

Menggandakan objek dalam pola tertentu

Langkah:
  1. Ketik AR → Enter
  2. Pilih objek, tentukan jenis dan pengaturannya
Shortcut: AR Modify

SCALE (SC)

Fungsi:

Memperbesar atau memperkecil ukuran objek dengan faktor skala tertentu.

Langkah:
  1. Ketik SC → tekan Enter
  2. Pilih objek yang ingin diubah → Enter
  3. Tentukan base point (titik acuan skala)
  4. Masukkan faktor skala → Enter
Shortcut: SC Modify

OFFSET (O)

Fungsi:

Membuat salinan objek sejajar pada jarak tertentu dari objek asli.

Langkah:
  1. Ketik O → tekan Enter
  2. Masukkan jarak offset → Enter
  3. Klik objek yang akan di-offset
  4. Klik di sisi arah offset (dalam/luar, kanan/kiri, atas/bawah)
Shortcut: O Modify

HATCH (H)

Fungsi:

Mengisi area tertutup dengan pola/tekstur

Langkah:
  1. Ketik H → Enter
  2. Pilih area tertutup atau klik titik dalam batas
  3. Pilih jenis hatch dan skala
Shortcut: H Draw

BLOCK (B)

Fungsi:

Menggabungkan objek menjadi satu kesatuan reusable

Langkah:
  1. Ketik B → Enter
  2. Beri nama block, pilih objek, tentukan base point
Shortcut: B Block

MATCH PROPERTIES (MA)

Fungsi:

Menyalin properti objek (warna, layer, line type) ke objek lain

Langkah:
  1. Ketik MA → Enter
  2. Klik objek sumber
  3. Klik objek target
Shortcut: MA Layers

DIMENSION (DIM)

Fungsi:

Memberi ukuran objek (panjang, sudut, radius, diameter)

Langkah:
  1. Ketik DIM → Enter
  2. Klik dua titik untuk memberi ukuran
Shortcut: DIM Annotation
